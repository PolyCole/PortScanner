# PortScanner
A port scanner written in Java using sockets.

<head>
  <main>
    <h3>Command parameters</h3>
    <ul>
      <li>-T  : Target IP address</li>
      <li>-to : Timeout (ms) </li>
      <li>-tc : Threadcount </li>
      <li>-p  : Number of ports to scan. </li>
      <li>-i  : Input filename </li>
      <li>-o  : Output filename </li>
    </ul>
  </main>
</head>
